# Product Features

> **TL;DR:** Comprehensive feature index covering behavioral analysis, vault system, Drift trading, privacy layer, and gamification.

## Core Features

- [[product/features/wallet-behavior-analysis]] — Pattern detection for revenge trading, overtrading, tilt
- [[product/features/trader-spec-cards]] — Visual trader profile and statistics display
- [[product/features/personalized-rule-recommendations]] — AI-driven rule suggestions based on history
- [[product/features/vault-system]] — Deposit, withdrawal, and rule configuration
- [[product/features/auto-lockout-system]] — Automated trading restrictions when limits breached
- [[product/features/emergency-unlock-system]] — Safety mechanisms for fund access

## Trading Features

- **Drift Perpetuals** — 31 perp markets via manual instruction builders
- **Trigger Orders** — Stop loss and take profit via ghost order system
- **Position Management** — Open, close, and modify positions

## Privacy Features

- [[product/features/privacy-layer]] — Privacy-preserving computation layer
- **Ghost Mode** — Private intent execution via MagicBlock Ephemeral Rollups
- **Shield Mode** — ZK-compressed P&L settlement via Light Protocol

## Gamification (New)

- **Trader Profiles** — XP, levels, and achievement system
- **Discipline Tracking** — Scores for patience, discipline, and emotional control
- **Streak Rewards** — Bonuses for consistent rule-following
